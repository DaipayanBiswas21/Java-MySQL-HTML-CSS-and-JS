
Planet Earth Monitoring System v2
----------------------------------
Features:
- Admin login: View, Insert, Update planet monitoring data.
- Normal user login: View only.
- Parameters: AQI, WQI, Temperature, Humidity, Noise Level, Latitude, Longitude, Date.
- Swing UI: Eye-catchy with gradient headers and custom fonts.
- JDBC for MySQL connection.
- MyException for custom error handling.
- OOP principles applied across all classes.

Setup:
1. Import the SQL from 'resources/schema.sql' to MySQL.
2. Update 'db.properties' with your MySQL credentials.
3. Compile and run using instructions in the README.

This is just a placeholder. In the actual implementation,
the Java files will be inside src/ with Swing and JDBC integration.
